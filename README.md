Project Title:-
Personal Portfolio Website

Project Overview
This project is a personal portfolio website designed to present professional information, skills, and projects in a clean and modern format. The website is fully responsive and works smoothly on desktop, tablet, and mobile devices. It provides a clear structure to showcase personal details, technical skills, project experience, and contact information.

The website is built using standard web technologies and follows best practices for layout, responsiveness, accessibility, and user experience.

Features:

Responsive navigation bar with mobile menu toggle,
Smooth scrolling between sections,
Modern hero (personal) section,
About section with professional summry,
Projects section with card layout,
Skills section displayed in a grid,
Contact form with basic validation,
Footer with social media links,
Scroll-based animations for better user experience,

Technologies Used:

HTML5 – Structure and content.
CSS3 – Styling, layout, animations, and responsiveness.
JavaScript – Interactivity and dynamic behavior.

Folder Structure:

How to Run the Project:

Download or clone the project files,
Open the project folder,
Double-click on index.html,
The website will open in your browser,
No additional setup or installation is required.
